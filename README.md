# Romagnoli Web Starter (Next.js + Tailwind)

Dos landings en un solo proyecto Next.js (App Router) para **Café Romagnoli** y **Toscano Tabaco**.

## Rutas
- `/` — Home del grupo
- `/caferomagnoli` — Landing de Café Romagnoli
- `/toscanotabaco` — Landing de Toscano Tabaco

## Cómo usar
```bash
npm install
npm run dev
# abre http://localhost:3000
```
